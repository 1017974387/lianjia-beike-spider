# LianJiaSpdier

链家爬虫，获取链家网小区房价数据。

* 城市 city
* 区县 district
* 板块 area
* 小区 xiaoqu
- 每个版块一个csv文件，默认采集上海市的二手房小区房价数据
- 内容格式：采集日期,所属区县,板块名,小区名,挂牌均价,挂牌数
- 内容如下：20180221,浦东,川沙,恒纬家苑,32176元/m2,3套在售二手房


changes: 
- 2018/02/21 应对链家前端页面更新，使用urllib2代替第三方requests库,提升性能，减少依赖
